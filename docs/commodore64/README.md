# Commodore 64 technical references

Retrieved: 2026-07-07

These files are local reference copies for Commodore 64 emulator development.
Zimmers notes that some manuals were originally distributed with Commodore
computers and are for own use if an original is missing; keep these copies for
local reference and check upstream notices before redistribution.
