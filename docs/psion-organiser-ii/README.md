# Psion Organiser II technical references

This directory contains local copies of technical reference material from
Jaap Scherphuis' Psion Organiser II archive:

https://www.jaapsch.net/psion/index.htm

Included files:

- `tech.htm` and `tech01.htm` through `tech21.htm`: Psion Organiser II
  Technical Manual.
- `techlz.htm`, `flash.htm`, `ionotes.htm`, `rsctrl.htm`, `topslot.htm`,
  `models.htm`, `fileform.htm`, `protocol.htm`, `qcodes.htm`,
  `sysvars.htm`, and `datasheets.htm`: related technical pages used for
  platform implementation and external connector behavior.
- `roms.htm`: source page for the Psion ROM images used by the emulator ROM
  catalog.
- `pics/topslot/*.gif`: top-slot pinout diagrams referenced by `topslot.htm`.

The original technical manual pages state Psion copyright. Keep these copies
for local emulator development/reference use and retain this provenance note.
