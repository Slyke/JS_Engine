Psion Organiser II Datasheets and Schematics

This is a collection of Datasheets for the various chips used in the Psion Organiser and its packs.

The Organiser's chips:

  • Datasheet HD6303X MPU (1.58MB pdf)
    Datasheet for the HD6303 processor, the MPU that is contained in the Psion Organiser Series II.
  • Datasheet HD6301X MPU (1.41MB pdf)
    Datasheet for the HD6301 processor, the MPU that is contained in the Psion Organiser Series 1.
  • Datasheet HD44100 LCD driver (67KB pdf)
    Datasheet for the HD44100, the display driver used in the Psion Organiser.
  • Datasheet HD44780 LCD controller (279KB pdf)
    Datasheet for the HD44780, the display controller used in the Psion Organiser.
  • HD6301/HD6303 Series Handbook (51.7MB pdf)
    Handbook for the HD6301/3 processors, including user manuals, software application notes, hardware application notes.

Pack chips:

  • Datasheet Eprom 8K chip, 27C64 (67KB pdf)
    Datasheet for the 8K Eprom chip that is contained in an 8K Psion datapak.
  • Datasheet Eprom 16K chip, 27C128 (55KB pdf)
    Datasheet for the 16K Eprom chip that is contained in a 16K Psion datapak.
  • Datasheet Eprom 32K chip, 27C256 (90KB pdf)
    Datasheet for the 32K Eprom chip that is contained in a 32K Psion datapak.
  • Datasheet Eprom 64K chip, 27C512 (70KB pdf)
    Datasheet for the 64K Eprom chip that is contained in a 64K Psion datapak.
  • Datasheet Eprom 128K chip, D27011 (441KB pdf)
    Datasheet for the 128K Eprom chip that is contained in a 128K Psion datapak.
  • Datasheet Flashpak 128K chip, 28F010 (338KB pdf)
    Datasheet for the memory chip that is contained in a 128K and 256K Psion flashpak.
