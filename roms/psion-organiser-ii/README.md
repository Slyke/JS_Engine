# Psion Organiser II ROM images

These ROMs were downloaded from Jaap Scherphuis' Psion Organiser II ROM page:

https://www.jaapsch.net/psion/roms.htm

That page states that Psion retains copyright on the ROMs but has given
permission for emulator use. Included images:

- `26-cm.rom`
- `26-xp.rom`
- `31-xp.rom`
- `46-lz.rom`

The default Psion Organiser II platform entry uses the LZ `46-lz.rom`.
The older two-line XP entry uses `31-xp.rom`.
